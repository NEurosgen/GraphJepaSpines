# Graph Jepa Spines



Базовая ветка экспериментов
От нее наследуются все остальные 
В дальнейшем тут будет обнавляться код который оказался полезным во многих запусках
Код в каждой из веток должен сожердать краткий отчет о целях эксперимента и его результатх.

Краткое описание существущих эуспериментов:
    enc_layers - Измерение зависимости val_loss от num_layers
    alpha_enc - Измерения зависимости val_loss от alha (res con)
    repr_mask - Измерение зависимости репрезентотивности по проценту маскировки
    vizualization - визуализация латентного пространства датсет 9009 по чекпойнту
    bathc_loss - Измерения зависимости val_loss от размера batcha
