# Graph Jepa Spines

Описание эксперимента:

Выяснить зависимость val_loss от числа слоев в энкодере (При условии что JEPA не схлопывается)

Код в run_experiments :
    Устанваливаем не изменяемыми все параметры , кроме num_layers (перед этим можно прикинуть их значения для адекватной работы , после чего фиксировать)
    Число эпох 10 
    Устанавливаем mask_ratio в очень маленькое значение (например 0.01 ) чтобы у модели точно была возможность для приближенного востонавления значений
    Пробегаем num_layers от 1 до 30 и записываем значения как min_val_loss так и val_loss на последней эпохе для построения графика. Для одного и того же значения num_layers проводим жксперимент 3 раза с разынми seed для стабильности

    Все записанные данные сохраняем в exp/enc_layers
