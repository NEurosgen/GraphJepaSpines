# Graph Jepa Spines

Описание эксперимента:

Выяснить зависимость val_loss от числа слоев в энкодере (При условии что JEPA не схлопывается)

Результаты:
 Лосс падает вплоть до 7 слоев потом медленно начинает рассти (проводил измерения на от 1 до 10 слоев)
 Возможно стоит посмотреть зависимость поведения фиксированного энкодера в зависимости от alpha 
 Если зависимость будет сильной стоит провести замеры по двум степеням свободы( слои и alpha)
